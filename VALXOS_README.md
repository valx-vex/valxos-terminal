## Quick Start

### Install Dependencies
```bash
# OpenCode (base framework) - already installed
brew install opencode

# Codex CLI (HAL 10000)
npm install -g @openai/codex

# Gemini CLI (2M context)
brew install gemini-cli

# Ollama (optional, local inference)
brew install ollama
ollama pull qwen3.5:27b
```

### Build Legion Plugin
```bash
cd ~/cathedral-prime/03-code/active/valxos-terminal/packages/legion
bun install
bun run build

# Link commands
sudo ln -s $(pwd)/bin/legion.mjs /usr/local/bin/legion
```

### Test Dispatch
```bash
# Auto-route
legion "Write a Python function to sort a list"

# Force backend
legion --backend codex --effort high "Review this code"

# Quick (FREE)
legion --quick "What is Python syntax for decorators?"
```
